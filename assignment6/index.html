DYNAMIC MULTIPLICATION TABLE

Please enter the starting and ending numbers for both the horizontal (multiplier) and vertical (multiplicand) axes and click submit to create the multiplication table.

The default values for Horizontal Start Number(1), Horzontal End Number (10) AND Vertical Start Number(1), Vertical End Number(10) create a 10 X 10 multiplication table that runs from 1 to 10 vertically and horizontally. Enter the numbers manually OR click the arrows (that "appear") on the right of each textbox to increae or decrease the current number
